🔐 Clés USB administrateur – Fonctionnement et gestion
L’application utilise un système de contrôle par clés USB administrateur. Ces clés permettent de sécuriser les actions sensibles, comme :

Désactiver le mode examen,

Accéder aux paramètres internes,

Ajouter ou supprimer d’autres clés administrateur.

🆔 Comment une clé est-elle reconnue ?
Chaque clé USB possède un numéro de série matériel unique. Ce numéro est automatiquement détecté lors de l’insertion, puis comparé à une liste enregistrée dans le fichier admin_key.csv.

Si le numéro correspond à une clé autorisée, les fonctions administrateur sont accessibles.

🔓 Accès aux fonctions administrateur
Pour accéder à des actions sensibles (paramètres, ajout/suppression de clés, désactivation du mode examen), au moins une clé administrateur autorisée doit être connectée.

⚠️ Si aucune clé autorisée n’est détectée, l’accès est bloqué.

➕ Ajouter une nouvelle clé administrateur
Pour enregistrer une nouvelle clé :

Connectez une clé administrateur déjà autorisée à l’ordinateur.

Connectez ensuite la nouvelle clé USB que vous souhaitez enregistrer.

Ouvrez le menu d’ajout de clés dans l’application.

Choisissez la nouvelle clé dans la liste proposée.

Donnez-lui un nom d’identification, puis validez.

La clé est alors ajoutée automatiquement au fichier de configuration avec son nom et son numéro de série.

⚠️ Important à retenir
L’ajout de nouvelles clés n’est possible que si une clé administrateur est déjà présente.
Ce système garantit que seul un utilisateur autorisé peut enregistrer de nouvelles clés, empêchant toute manipulation non autorisée.
