1) Nouvelles fonctionnalités ajoutées :

- Si le blocage du presse-papier est activé, son contenu est remplacé toutes les secondes, empêchant ainsi toute tentative de copier-coller depuis des éditeurs de texte comme les fichiers Notes, Word, ou tout autre fichier existant, même créé avant l'épreuve.

- Un bouton "Aide" a été ajouté pour fournir un guide simple et rapide sur le fonctionnement de l'application.

2) Suppression de certaines fonctionnalités :

- Le bouton, "Informations" à destination de l'élève examiné, a été retiré, le message s'affiche désormais après lancement du mode examen. Ce message explique brièvement le fonctionnement de l'application, et la procédure à suivre si son ordinateur venait à être verrouillé.

3) Modifications de l'interface :

- Une barre d'informations a été intégrée pour afficher l'état de la batterie, la durée du mode examen activé, le nombre de clés enregistrées, ainsi que le nombre d'applications bloquées. (Note : le compteur d'applications bloquées peut actuellement afficher des valeurs erronées, ce qui sera corrigé dans une future version.)

4) Correction de bugs : 

- Vérification d'une nombre de clé USB administrateur, avant la supression d'une clé, bloquant ainsi la supression s'il n'y a qu'une seule clé.
